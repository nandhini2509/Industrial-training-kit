# AI Data Classification using Decision Tree

## Project Description
This project is an AI-based data classification system developed using Python and Machine Learning.

The model uses a Decision Tree Classifier to classify data into different categories based on input features.

## Features
- Data preprocessing using Pandas
- Dataset splitting into training and testing sets
- Decision Tree Classification
- Model accuracy evaluation
- Model saving using Pickle

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Pickle

## Files Included
- project.py
- dataset.csv
- model.pkl
- code_screenshot.png
- output_screenshot.png

## How to Run
1. Install required libraries:
   pip install pandas scikit-learn

2. Place dataset.csv in the project folder.

3. Run the program:
   python project.py

## Output
- Displays model accuracy.
- Saves trained model as model.pkl.
